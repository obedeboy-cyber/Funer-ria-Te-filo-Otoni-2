# Site Funerária Teófilo Otoni

Projeto React + Vite + Tailwind pronto para publicar na Vercel.

## Rodar localmente

```bash
npm install
npm run dev
```

## Publicar na Vercel

- Framework Preset: `Vite`
- Root Directory: `./`
- Build Command: `vite build`
- Output Directory: `dist`
- Install Command: `npm install`
- Environment Variables: deixe vazio
- Deploy

## Observações

- A logo está em `public/logo.png`
- A imagem da família está em `public/family.webp`
- O código principal está em `src/App.jsx`
